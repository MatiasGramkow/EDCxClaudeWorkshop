# Survey Responses

Gem svarene fra workshoppen her som JSON-filer.

Når du har modtaget emails fra formsubmit.co, kopier JSON-dataen fra
`_JSON_DATA`-feltet ind i en fil her, f.eks. `anders-jensen.json`.

Kør derefter Claude Code i denne mappe og bed den analysere svarene:

```
claude "Læs alle JSON-filer i responses/ mappen og giv mig et overblik over
holdets niveau, hvad de vil lære, og hvad jeg bør fokusere på i workshoppen"
```
